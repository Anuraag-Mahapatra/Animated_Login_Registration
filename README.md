# Animated Login & Registration Page

A sleek, responsive login/registration interface with **smooth CSS animations** and seamless form transitions. Toggle between login and registration with single-click links—professional UX without frameworks.

## ✨ Features
- **Fluid Transitions**: CSS-powered slide animations between forms
- **Responsive Design**: Perfect on desktop, tablet, mobile (flexbox + media queries)
- **Interactive Elements**: Hover effects, focus states, input validation styling
- **Modern Styling**: Gradient backgrounds, glassmorphism shapes, BoxIcons
- **Vanilla Implementation**: Pure HTML/CSS/JavaScript—no dependencies

## 🛠️ Tech Stack
Frontend: HTML5 | CSS3 (Flexbox, Animations, Gradients)
Icons: BoxIcons CDN
JavaScript: DOM Manipulation

## 🚀 Live Demo
Open `index.html` in any modern browser. No setup required!

## 📁 File Structure
├── index.html # Structure + forms
├── style.css # Animations, responsive design, effects
└── script.js # Form toggle logic

## 🎨 Key Animations
- **Form Slide**: `.current` class triggers 1.5s staggered transitions
- **Shape Morphing**: Background shapes rotate/skew on state change
- **Input Focus**: Labels animate upward with color transitions
